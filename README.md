## <p style ="color: #8A90FA;">ABOUT THIS PROJECT</p>
**Cash Back Interactive** is a beginner C# project built as part as my C# programming course. The app takes a dollar amount entered by the user and calculates how to break it into the fewest possible bills (specifically **$20s, $10s, $5s, and $1s**).

This project helped me practice:
- Windows Form UI design in Visual Studio
- Division and modulus operators in C#
- Event-driven programming (button click handlers and key event handlers)
- Basic input validation 

> *This is a learning assignment. The goal was to learn the fundamentals, not to build production software.*

## <p style ="color: #8A90FA;">HOW TO USE </p>
1. **Launch** the application by running "CashBackInteractive.exe" (or open the ".sln" in Visual Studio and press *F5* to run)
2. **Enter** a whole dollar amount into the input field (ex: "27")
3. **Click** the *SUBMIT* button to display the bill breakdown
4. **Click** the *CLEAR* button to remove previous responses
